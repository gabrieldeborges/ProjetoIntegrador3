Guia para uso da aplicação

Feito por Gabriel Borges e Lucas Filipe Chaves
	Nosso projeto usa a base de dados do governo que armazena informações sobre os Centros Esportivos de São Paulo. 
	Em cada centro existe alguns locais, como quadra, campos, piscinas, entre outros. Baseado nisso, decidimos fazer uma aplicação na qual você alugue esses espaços para realizar suas atividades esportivas. 
	Para isso você deve se cadastrar, então escolher o tipo de equipamento que você deseja alugar e por último confirmar o seu aluguel.
Passo 1 O projeto será iniciado na tela de login, caso você já tenha cadastro, basta apenas inserir os dados que você entrará, caso não tenha, clique no botão de cadastrar. Você será redirecionado para a tela de cadastro.
Passo 2  Após realizar seu cadastro, você vai entrar e pode escolher algum espaço para ser alugado. Clique em qual espaço deseja.
Passo 3 Depois de selecionado, você será redirecionado para a tela de confirmação de aluguel, lá você pode pesquisar pelo CEP e vai aparecer unidades próximas. Selecione a data e confirme.
Passo 4 Quando você logar novamente, você poderá avaliar o seu uso. 
Há uma bibliotecas a serem usadas, estão armazenadas na pasta “libs”.
O MySQL Connector e o GSON.
